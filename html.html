Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web.

Ejemplo de código:


<!DOCTYPE html>
<html>
<head>
<title>Mi Página Web</title>
</head>
<body>
<h1>¡Hola, mundo!</h1>
</body>
</html>

Este código genera una página con un título y un encabezado.

Elementos básicos de HTML

Párrafo


<p>Mi primer párrafo.</p>

Agrega contenido textual usando la etiqueta <p>.

Encabezados


<h1>Título Principal</h1>
<h2>Subtítulo</h2>

Los encabezados estructuran el contenido por niveles de importancia.

Enlaces


<a href="url">Texto del enlace</a>

Los enlaces dirigen al usuario a otras páginas o recursos.

Imágenes


<img src="url" alt="Descripción de la imagen">

Usa imágenes para mejorar el atractivo visual de la página.
