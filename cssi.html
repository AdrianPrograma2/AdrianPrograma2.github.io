Introduction to CSS

CSS describes how HTML elements should be displayed on screens, paper, or other media.

Change text color and alignment


p {
    color: red;
    text-align: center;
}

This code allows you to change the color of paragraphs and center them on the screen.

Set background color


body {
    background-color: lightblue;
}

This code changes the background color of the webpage.

Use an image as background


body {
    background-image: url("paper.gif");
}

This code sets an image as the background of the webpage.

Add margins


p {
    margin-top: 100px;
    margin-bottom: 100px;
    margin-right: 150px;
    margin-left: 80px;
}

This code adds margins around elements to improve the layout.

Set height and width


div {
    height: 200px;
    width: 50%;
    background-color: powderblue;
}

This code defines the height and width of an element with a specific background.
