Introducción a CSS

CSS describe cómo se deben mostrar los elementos HTML en la pantalla, el papel o en otros medios.

Cambiar color y alineación de texto


p {
    color: red;
    text-align: center;
}

Este código permite cambiar el color de los párrafos y centrarlos en la pantalla.

Cambiar el color de fondo


body {
    background-color: lightblue;
}

Este código cambia el color de fondo del cuerpo de la web.

Usar una imagen como fondo


body {
    background-image: url("paper.gif");
}

Este código establece una imagen como fondo de la web.

Añadir márgenes


p {
    margin-top: 100px;
    margin-bottom: 100px;
    margin-right: 150px;
    margin-left: 80px;
}

Este código añade márgenes alrededor de los elementos para mejorar el diseño.

Establecer altura y ancho


div {
    height: 200px;
    width: 50%;
    background-color: powderblue;
}

Este código define la altura y el ancho de un elemento con un fondo específico.
