Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

Code Example:


<!DOCTYPE html>
<html>
<head>
<title>My Web Page</title>
</head>
<body>
<h1>Hello, world!</h1>
</body>
</html>

This code generates a page with a title and a heading.

Basic HTML Elements

Paragraph


<p>My first paragraph.</p>

Add text content using the <p> tag.

Headings


<h1>Main Title</h1>
<h2>Subtitle</h2>

Headings structure content based on importance levels.

Links


<a href="url">Link text</a>

Links direct users to other pages or resources.

Images


<img src="url" alt="Image description">

Use images to make your web page more visually appealing.
